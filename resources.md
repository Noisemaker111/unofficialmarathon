## 1. Official Resources

### Aleph One (Open-Source Marathon Engine)

| Property | Detail |
|---|---|
| **Repository** | [github.com/Aleph-One-Marathon/alephone](https://github.com/Aleph-One-Marathon/alephone) |
| **Stars** | 770+ |
| **Language** | C++ |
| **Website** | [alephone.lhowon.org](https://alephone.lhowon.org) |
| **Latest Release** | v1.11b (August 2025) |

**Features**: Cross-platform (macOS, Windows, Linux), OpenGL shaders, Lua scripting, Marathon 2 film playback, network multiplayer.

**GitHub Data Repos**:
- [data-marathon](https://github.com/Aleph-One-Marathon/data-marathon) — Marathon 1 game files
- [data-marathon-2](https://github.com/Aleph-One-Marathon/data-marathon-2) — Marathon 2 game files
- [data-marathon-infinity](https://github.com/Aleph-One-Marathon/data-marathon-infinity) — Infinity game files

### Marathon Source Code Releases

- **Marathon 2 source** — Released January 2000 ([archive on GitHub](https://github.com/Aleph-One-Marathon/m2-infinity-source-code))
- **Marathon Infinity source** — Released Bungie Day 2011 ([infinitysource.bungie.org](http://infinitysource.bungie.org/))
- **Aleph One 1.0** — December 1, 2011; free remastered release of all three games ([Eurogamer coverage](https://www.eurogamer.net/articles/2011-12-02-remastered-marathon-games-released-free-on-pc-mac-linux))

### Marathon Trilogy Box Set (1997)

Physical box set containing all three games plus Forge & Anvil editors and PDF manuals.
- Download: [trilogyrelease.bungie.org](https://trilogyrelease.bungie.org/)

Developed by Freeverse Software. Released August 2007 on Xbox 360.

### Community Memes & Catchphrases

| Phrase | Origin | Meaning |
|---|---|---|
| **"Frog blast the vent core!"** | Said by Simulacrum (BOB simulacrum) in Marathon 2: Durandal. Coined by Doug Zartman during early development. | The quintessential Marathon quote; used as a community shibboleth. Acronym: FBTVC. |
| **Vidmaster** | Jason Jones' 1995 challenge post | Title for completing Marathon under extreme conditions |

### Vidmaster Challenge Traditions

- Completing games under specific difficulty/time constraints
- Recording and sharing runs
- Recognition on Bungie forums and community sites
- The challenge spawned an entire speedrun community

### Community Download Hub

**Simplici7y** ([simplici7y.com](https://simplici7y.com)) — User accounts, ratings, reviews, tags (scenario, textures, shapes, monsters, HD, solocoop). Most downloaded: Marathon Phoenix, Excalibur: Morgana's Revenge.
